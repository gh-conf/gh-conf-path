## Install

```
$ npm install @gh-conf/gh-conf-path
```

## Usage

```javascript
const { addSlash } = require('@gh-conf/gh-conf-path');

const path = addSlash('./gh-conf-path');
console.log(path)
// Output
// ./gh-conf-path/

```
